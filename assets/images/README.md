# Image assets

Drop your real image files into this folder using these exact names — every
page already references them, so nothing else needs to change once they're here.

| Filename | Used on | Notes |
|---|---|---|
| `logo-icon.png` | Header (all pages), favicon | Hexagon mark only, square, transparent background recommended |
| `logo-full.png` | Footer (all pages) | Full lockup with wordmark |
| `product-photo.jpg` | Home hero, Product hero | Landscape, roughly 4:3 |
| `office-photo.jpg` | Contact page | Landscape, roughly 4:3 |

Until a file is added, each page shows a graceful placeholder (a simple flame
mark for the logo, a labeled navy box for photos) so the layout still looks
correct.
